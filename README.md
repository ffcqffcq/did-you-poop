# 拉了么

“拉了么”是一个离线优先的 Android 记录型应用，用来记录每天如厕次数、单次时长，并支持每日定时提醒和周/月数据分析。

## 当前 MVP

- 手动记录一次如厕，并填写本次时长和备注
- 计时记录，结束后自动写入一条记录
- 今日概览：次数、总时长、单次平均时长
- 近 7 天 / 近 30 天分析：日均次数、平均时长和基础健康提示
- 每日定时提醒，设备重启后自动恢复提醒
- 最近 10 条记录展示和删除
- 补录过去的日期和时间
- 编辑已有记录
- 记录形态和是否有明显不适
- 每日趋势列表
- 日历选日期查看当天完整记录
- 从所选日期直接补录
- 底部导航将记录、分析、日历、提醒分区展示
- 顶部安全区适配，减少被摄像头或状态栏遮挡
- 开屏页和可爱便便贴图
- 首次安装或版本更新后的第一次进入会弹窗说明软件用途
- 记录数据保存在本机 SQLite，提醒设置保存在 SharedPreferences，不需要网络和账号

## 后续规划

1. 增加记录维度，例如饮水量、膳食纤维、运动量。
2. 增加趋势图、异常连续天数判断和更细的提示规则。
3. 增加数据导出、备份和恢复。
4. 增加隐私保护，例如应用锁或本地加密。
5. 拆分多页面导航，让首页更轻、更适合长期使用。

## 构建方式

本项目是标准 Android Studio 工程，可以用 Android Studio 打开项目并运行 `app` 模块，也可以在 Windows 终端运行：

```powershell
.\gradlew.bat :app:assembleDebug
```

推荐环境：

- Android Studio Ladybug 或更新版本
- JDK 17
- Android SDK 35
